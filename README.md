# How to Start this engine

there are two methods to start this engine:
one is by running the engine manually and the other is by running the engine in a docker container cluster.

1. Running the engine manually
-  configure a virtual environment for python ``` python -m venv .venv ```
-  install the required dependencies ``` uv sync ```
-  run the engine ``` python app.py```

2. Running the engine in a docker container cluster
- just run ```docker-compose up ```
